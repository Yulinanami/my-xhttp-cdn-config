# XHTTP + CDN 搭建指南 (三)：xpadding 配置

本文档是在 [2.文件配置.md](./2.文件配置.md) 的基础上，补充 **xpadding** 相关配置。

---

## 1. 版本要求

- **Xray 内核版本**：≥ `26.2.6`
- **Mihomo 内核版本**：≥ `1.19.24`

推荐直接使用较新的稳定版本。

---

## 2. 推荐的 xpadding 写法

这套是当前仓库实际验证可用的保守写法：

- `xPaddingObfsMode: true`
- `xPaddingKey: "x_padding"`
- `xPaddingHeader: "Referer"`
- `xPaddingPlacement: "queryInHeader"`
- `xPaddingMethod: "tokenish"`

对应到 Mihomo 写法则是：

- `x-padding-obfs-mode: true`
- `x-padding-key: "x_padding"`
- `x-padding-header: "Referer"`
- `x-padding-placement: "queryInHeader"`
- `x-padding-method: "tokenish"`

---

## 3. Xray 服务端配置

编辑 `/usr/local/etc/xray/config.json`，找到 **XHTTP 入站** 的 `streamSettings.xhttpSettings`，在原本的：

```json
"xhttpSettings": {
  "host": "",
  "path": "/cdn",
  "mode": "auto"
}
```

基础上改成：

```json
"xhttpSettings": {
  "host": "",
  "path": "/cdn",
  "mode": "auto",
  "xPaddingObfsMode": true,
  "xPaddingKey": "x_padding", // 可自定义修改，注意后续相同配置项的值要一致
  "xPaddingHeader": "Referer", // 可自定义修改，注意后续相同配置项的值要一致
  "xPaddingPlacement": "queryInHeader",
  "xPaddingMethod": "tokenish"
}
```

- 这里改的是 **XHTTP 入站**
- `Reality + Vision` 那个直连入站**不用加**
- 这部分是 **Xray 配置字段**，不要写成 Mihomo 的 `x-padding-*`

## 5. Mihomo 客户端

在 `xhttp-opts` 里增加下面 5 个字段：

```yaml
x-padding-obfs-mode: true
x-padding-key: "x_padding"
x-padding-header: "Referer"
x-padding-placement: "queryInHeader"
x-padding-method: "tokenish"
```

### 5.1 节点 2：xhttp+Reality 上下行不分离

```yaml
xhttp-opts:
  path: "YOUR_PATH"
  mode: auto
  x-padding-obfs-mode: true
  x-padding-key: "x_padding"
  x-padding-header: "Referer"
  x-padding-placement: "queryInHeader"
  x-padding-method: "tokenish"
  reuse-settings:
    max-concurrency: "16-32"
    c-max-reuse-times: "0"
    h-max-reusable-secs: "1800-3000"
    h-keep-alive-period: 0
```

### 5.2 节点 3 / 5：带 download-settings 的节点

除了外层 `xhttp-opts` 要加，同样也要在 `download-settings` 里补上：

```yaml
download-settings:
  path: "YOUR_PATH"
  # 其它字段按你原有配置填写
  x-padding-obfs-mode: true
  x-padding-key: "x_padding"
  x-padding-header: "Referer"
  x-padding-placement: "queryInHeader"
  x-padding-method: "tokenish"
  reuse-settings:
    max-concurrency: "16-32"
    c-max-reuse-times: "0"
    h-max-reusable-secs: "1800-3000"
    h-keep-alive-period: 0
```

### 5.3 节点 4：xhttp+TLS 双向 CDN

和节点 2 一样，在外层 `xhttp-opts` 加上同样的 `x-padding-*` 字段即可。
